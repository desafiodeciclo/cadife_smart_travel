## 2. Glossário de Termos

| Termo | Definição |
|---|---|
| **RAG** | Retrieval-Augmented Generation — técnica para reduzir alucinações via base de conhecimento. |
| **Webhook** | Endpoint que recebe notificações automáticas (ex: Meta/WhatsApp). |
| **Lead** | Potencial cliente que iniciou contato e foi registrado no sistema. |
| **Briefing** | Conjunto de dados da viagem (destino, datas, etc.) coletado pela IA. |
| **Curadoria** | Atendimento personalizado profundo realizado por humanos. |
| **FCM** | Firebase Cloud Messaging — serviço de notificações push. |
| **JWT** | JSON Web Token — padrão para autenticação segura entre App e API. |
| **Score** | Classificação da temperatura do lead (Quente, Morno, Frio). |
| **LangChain** | Framework para orquestração de cadeias de IA e LLMs. |
| **Vector DB** | Banco de dados para busca semântica (ChromaDB ou PGVector). |
